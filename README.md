# Update CSS Variables with JavaScript 🎨

This project demonstrates how to dynamically update CSS variables using JavaScript.

## 🚀 Features
- Adjust spacing, blur, and base color in real-time
- Interactive UI with range and color inputs
- Uses CSS variables (`:root`) for dynamic styling
- Event listeners in JavaScript for live updates

## 🛠️ Tech Stack
- HTML
- CSS (CSS Variables)
- JavaScript (DOM Manipulation)

## 📌 What I Learned
- How to use CSS custom properties (`--variables`)
- Updating styles dynamically using JavaScript
- Handling input events and dataset attributes
